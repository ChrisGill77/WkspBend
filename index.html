Sheet Metal Bending Help
This app performs the calculations required for bending sheet metal in a bending break. This is not intended for use with industrial benders.

Notes on the calculation

The calculation relies on a K value, which is determined by the properties of the metal, its thickness and the bending method. Because there are no standard values, workshops often have their own tables that work with their methodology and materials. There are, however, tables of typical values for soft (aluminium), medium (mild steel) and hard (other non-brittle alloys).

The app lets the user select a soft, medium or hard metal and either provide a K value or leave it empty. In the latter case the app will use a value from its built-in table (see below).

The Setup button opens a second page where the user can edit the table of K values.

Setup Help
This page allows default K values to be specified. These are arranged by type of metal (soft, medium or hard) and the ratio of the bending radius to the metal thickness. Leave the R/T ratio empty on the last row to account for ratios greater than on the penultimate row.

The number of decimals on the main page can also be set (the settings page always shows two decimals).

All settings are stored locally and re-loaded when the app opens.
